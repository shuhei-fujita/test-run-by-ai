# ログイン機能

ユーザーがアプリケーションにログインできることを確認する

## Scenario: 有効な認証情報でログインする

1. ブラウザで "$TEST_BASE_URL/login" を開く
2. メールアドレス欄に "$TEST_USER_EMAIL" を入力する
3. パスワード欄に "$TEST_USER_PASSWORD" を入力する
4. ログインボタンをクリックする
5. ダッシュボードページが表示される
6. 認証状態を "auth.json" に保存する

## Scenario: 保存済みの認証状態でログインをスキップする

1. 認証状態 "auth.json" を読み込む
2. ブラウザで "$TEST_BASE_URL/dashboard" を開く
3. ログイン画面にリダイレクトされずダッシュボードが表示される
