# YouTube動画検索

YouTubeで検索機能が正しく動作することを確認する

## Scenario: トップページから動画を検索する

1. ブラウザで "https://www.youtube.com/" を開く
2. 検索ボックスに "Playwright tutorial" と入力する
3. Enterキーを押して検索を実行する
4. 検索結果ページに遷移する
5. 動画のサムネイルが一覧表示される
6. スクリーンショットを撮影する

## Scenario: 検索結果から動画ページに遷移する

1. ブラウザで "https://www.youtube.com/" を開く
2. 検索ボックスに "Playwright tutorial" と入力する
3. Enterキーを押して検索を実行する
4. 検索結果の最初の動画をクリックする
5. 動画再生ページに遷移する
6. 動画タイトルが表示される
7. スクリーンショットを撮影する
